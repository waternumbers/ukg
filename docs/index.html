Flood Forecast

The Flood Forecast website is an ongoing development by waternumbers to provide an opensource flood forecasting system for the United Kingdom based on publically available, redistributable data.

The project is under development and should not be seen as replacement for the flood warnings provided by the National agencies which are available at:

  • England: https://check-for-flooding.service.gov.uk/alerts-and-warnings
  • Scotland: https://floodline.sepa.org.uk/floodupdates/
  • Wales: https://flood-warning.naturalresources.wales/
  • Northern Ireland: https://www.nidirect.gov.uk/articles/check-risk-flooding-your-area

Status

Flood Forecast is currently at an early stage of development. Currently it provides

  • A viewer allowing visualisation of the available data
  • An API serving observed and forecast data
  • Background information in the form of presentations.
  • A github repository of the code used.

Given the early stage fo the project features such as the API are likely to undergo breaking changes

If you wish to make use of, help develop or fork this project please contact us either via raising an issue on github or by emailing contact@waternumbers.co.uk.
